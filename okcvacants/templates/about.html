About OKC Vacants

OKC Vacants is a mapping web app containing information on properties on the City of Oklahoma City's Abandoned Buildings list. This website is not affiliated with the City of Oklahoma City; all I do is take the vacant property data posted by the City and turn it into a map.

Reporting Properties

If you would like to report properties that should (or should not) be on the abandoned properties registry, I would suggest contacting the City's Action Center. The map will be updated after the abandoned properties list on the City's website is updated.

Coverage Area

Please note that the map only covers abandoned properties within the incorporated city limits of Oklahoma City. These city limits are shaded in light gray. This does not include areas in municipalities bordering or within Oklahoma City, nor does it include unincorporated areas. However, it may include areas of Oklahoma City with postal addresses in areas like Edmond or Yukon.

Technical Stuff

OKC Vacants is a custom web app. The backend was built with the Django web application framework and Python programming language and a Postgres/PostGIS database. The PyPDF2 library is used to parse the abandoned properties list from the City of OKC. Frontend was built using Bootstrap (for layout), Leaflet (for the map), and the jQuery library.

The source code for this app is available on GitHub.
